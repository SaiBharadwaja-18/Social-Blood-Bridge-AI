# Social-Blood-Bridge-AI
🩸 Autonomous AI-powered blood coordination platform for Thalassemia patients. Built with React, FastAPI, AWS Lambda, DynamoDB &amp; Amazon Bedrock Claude AI. Features real-time donor matching, predictive transfusion cycle triggers, and AI-generated donor outreach. Built for AI For Good 2.0 Hackathon by Blend360 × Blood Warriors x Hackculture.
